<h3> Descriptions </h3>
       <p> 
<b> 1. Content layer (HTML)</b> - This layer defines and lays out the text, images, animation, sound, video, and everything in the webpage using tags.  This is the underlying HTML code of a webpage. It is the foundation of the Website. 
<br>
<b> 2. Presentation Layer (CSS)</b> - This layer is how people see the web page in a browser. It must have interactive elements that are easy to find, and the images must have enough contrast to have an attractive webpage.
<br>
<b> 3. Behavior Layer (JavaScript)</b> - This layer is where you have a real-time user interaction, for example, filling out a form.
